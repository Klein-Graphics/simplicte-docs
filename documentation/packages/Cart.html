    Ajax Call To Add Item

    add_item(string $item_num) : null

    Ajax Controller Function Adds an item to the current customers cart. Echos the "Item Added" flag or an alternate flag that alters the customer that there item couldn't quite be added the way they wanted it. Item number is passed via the URI. All other (variable) data is posted to the function.
