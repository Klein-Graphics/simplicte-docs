\Library\Cart->add_item:
add_item($cart,$item,$options='0',$qty=1,$implode=NULL,$update=NULL)

if $item is an array, then all options after are ignored

if $cart is an array, then Implode and Update defaults to FALSE

if $cart is numeric, then Implode defaults to True and Update defaults to the cart ID

if $cart is a string, then Implode defaults to True and Update defaults to false
