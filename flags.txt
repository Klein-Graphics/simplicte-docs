Simplecart Flags

ITEMS

hide_qty
Hides the quantity

extscript:[external_script]
Causes an external script to be run after purchasing an item.
Can be used to alert external systems that an item has been bought.

min
Sets a minumum quantity

OPTIONS

template:[template]
Use a different template for this option

allow_qty (single only)
Allows a quanity box for the option

req
Makes the option required

no:[option_number]-[option_number]-...:[replacement]
Makes sure that this item isn't used with any of the [option_number]s and
if it is it's replaced with [replacement]
